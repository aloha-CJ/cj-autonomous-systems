# TextNow Signup Process

**Goal:** Get free permanent phone number for 2FA

## Steps
1. Go to https://www.textnow.com/signup
2. Sign up with ai@alohareefgear.com
3. Get assigned a free U.S. phone number
4. Verify email (check ai@alohareefgear.com inbox)
5. Note phone number for future use
6. Test 2FA capability

## Automation Plan (After Manual Setup)
- Install PyTextNow: `pip install PyTextNow`
- Automate SMS retrieval for 2FA codes
- Use for service signups
